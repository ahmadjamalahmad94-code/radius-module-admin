# HobeRadius — صفحة الهبوط العامة

صفحة تسويقية عامة لمنتج **HobeRadius** (منصة تشغيل وإدارة شبكات الإنترنت و RADIUS).

## المسار
- `GET /` — الصفحة الرئيسية العامة. لو كان المستخدم مدير/عميل مسجّل دخول يُحوَّل للوحته، وإلا تُعرض صفحة الهبوط.
- تُرسم بالكامل من قاعدة البيانات عبر الـ CMS (انظر `LANDING_PAGE_CMS.md`).

## الملفات
| الملف | الغرض |
|------|------|
| `app/templates/public/landing.html` | القالب الديناميكي (يقرأ الأقسام/العناصر من DB، fallback آمن) |
| `app/static/css/public_landing.css` | هوية teal/indigo معزولة بكلاسات `.lp-*` (لا تؤثر على لوحة الإدارة) |
| `app/static/js/public_landing.js` | قائمة الموبايل + تنقّل سلس + أكورديون الأسئلة |
| `app/services/landing_cms.py` | seed المحتوى الافتراضي + دوال القراءة للعرض العام |

## الهوية البصرية
- ألوان: teal `#0d9488` + indigo `#4f46e5` (تدرّج هادئ)، خلفية off-white `#f6f8fa`، بطاقات بيضاء، حواف ناعمة، زوايا 18–24px.
- عربي RTL أولًا، خط Cairo، أيقونات FontAwesome 6.5.2، responsive (قائمة موبايل).
- معزولة تمامًا عن ستايل الإدارة `.hub-*` البنفسجي.

## الأقسام (افتراضيًا، كلها قابلة للتعديل من الإدارة)
Hero · شارات الثقة · مؤشرات · المزايا · الوحدات (مجمّعة) · التكاملات · طريقة العمل · خدمة واتساب · لماذا HobeRadius · الأسئلة الشائعة · دعوة للتواصل · الفوتر.

## دقّة التسويق (مهم)
المحتوى الافتراضي **يسوّق فقط الميزات المنفّذة فعلًا**: إدارة المشتركين، كروت Hotspot (استيراد CSV/XLSX/PDF)، الباقات/السرعات، MikroTik، RADIUS، مراقبة الشبكة، النسخ الاحتياطي + Google Drive، الدعم/التذاكر، التقارير، بوابة العميل، لوحة التراخيص، معالج الإعداد، REST API، Telegram، SMS، وWhatsApp Business Cloud API (متاح حسب الخطة).

**مستثنى عمدًا** (غير منفّذ/غير دقيق): «14 بوابة دفع»، PayMob/Fawry/InstaPay/PayPal/Stripe، UltraMsg، QR/رقم واتساب مشترك، ادّعاء Meta BSP، Gemini، Messenger bot، Firebase، ISO جاهز، One-click update.

خدمة واتساب موصوفة بأمانة: **العميل يستخدم رقمه التجاري ويدفع Meta مباشرة، وHobeRadius يوفّر طبقة الإدارة والأتمتة.**

## الاختبارات
`tests/test_landing_cms.py` — 14 اختبارًا تغطّي العرض من DB، الإخفاء/الترتيب، روابط التواصل/طرق الاتصال، حماية الإدارة، idempotency، غياب الادعاءات الممنوعة، وRTL.
