قواعد نفق WireGuard الإداري (wg-mgmt)

  • سرعة محدودة: 1–2 Mbps فقط (للبيانات الخفيفة)
  • لا Default Route — لا يمر عبره أي ترافيك إنترنت
  • لا NAT — لا Forward
  • AllowedIPs: عناوين الإدارة فقط (لا 0.0.0.0/0)
  • Firewall: يسمح منافذ API و Telemetry فقط
  • لا يُستخدم أبدًا لنقل بيانات العملاء

خدمة WireGuard البيانات (wg-data) تُنشأ كـ interface/service منفصل مع عداداته وحدوده المستقلة.
